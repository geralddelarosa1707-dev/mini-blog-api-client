# Mini Blog API Client (Python CLI)
This is a simple command-line application written in Python that interacts with the JSONPlaceholder API. It allows users to view, search, create, update, and delete blog posts through a menu-based interface.

## Features
• View all posts
• View a post by ID
• Search posts by title
• Create, update, and delete posts
• Logging system for actions and errors

## Technologies Used
- Python
- Requests library
- Python logging module

## API Used
This project uses the testing API from JSONPlaceholder to practice working with REST APIs.

## How to Run
• Install the required library:
- pip install requests
• Run the program:
- python main.py

## Purpose
This project was created to practice working with APIs, organizing Python projects into multiple files, and building a simple CLI application.
